# InlämningsUppgiftFullStackApplikation

Det här är en fullstack applikation som hanterar uppgifter eller tasks.
Frontend är byggt med HTML, CSS och JavaScrip som kommunicerar med ett.net web api 
backend som är byggt med C# och Entity Framework Core. Backend hanterar databasoperationer 
och affärslogik, medan frontend ansvarar för användargränssnittet och interaktionen med användaren.

Bland funktionerna i applikationen ingår att skapa, läsa, uppdatera och ta bort uppgifter (CRUD-operationer).
Det var en lärorikt uppgift fastän jag inte hade så mycket erfarenhet av webbutveckling innan. 

projektet körs genom att starta både frontend och backend samtidigt. Backend körs på en lokal server och frontend kan nås 
via webbläsaren. Frontend använder fetch API för att göra HTTP-förfrågningar till backend och uppdaterar användargränssnittet 
baserat på svaren från servern.

Jag lärde mig gå steg för steg med hjälp av AI och det var en rolig process att se hur allt kom samman i slutändan. 
Jag känner att jag har fått en bättre förståelse för hur frontend och backend kommunicerar med varandra 
fastän det är ändå svårt för mig att förstå alla koncept och tekniker som används i webbutveckling. 
men jag ser fram emot att fortsätta lära mig mer och förbättra mina färdigheter inom detta område.
